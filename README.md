# Linux Conversion Expert
## What is Linux Conversion Expert?
Linux Conversion Expert is a multiplatform, free, and open-source application designed to help people switch to Linux. It will do this by

- Helping users find software to use on Linux and helping them install it
- Teaching users how to install a Linux distro
- Finding online support if needed

## Why Linux?

Linux is fully free and open source. It works on almost any computer, so you can continue using your old hardware with is. Linux also doesn't promote other products, watch your usage, or bother you with AI and Onedrive.

With Linux, you are no longer bound to using Microsoft or Apple products and you don't constantly get bothered with updates.

The question should be: Why not Linux?

## What platforms will Linux Conversion Expert be available on?

- macOS
- Windows
- Linux
- Android
- Web

## What technologies were used?

- .NET (C#)
- Avalonia
